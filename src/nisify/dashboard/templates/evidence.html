Submit Manual Evidence

For controls requiring manual evidence (policies, audit reports, training records), use the CLI:

nisify submit --control GV.PO-01 --type policy_document \
  --url "https://sharepoint/security-policy.pdf" \
  --description "Information Security Policy v3.0"

Or submit a file directly:

nisify submit --control GV.RR-01 --type board_minutes \
  --file board-meeting-q4.pdf \
  --description "Q4 2024 Security Review"

List available evidence types:

nisify submit --list-types
